# PHP FPM 8.2 based on Alpine 3.19

## Utilisation

Pour utiliser l'image depuis le registre public :
```bash
docker pull rg.fr-par.scw.cloud/registry-par-ixys/containers/web/php-fpm-8.2:latest
```
